# API Protection Demo

This is a demo project using Azion to show how much an API with database requests can cost and the risk of not having it protected.

## Running the project

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Run the development server:

   ```bash
   pnpm dev
   ```

3. Build the project:

   ```bash
   pnpm build
   ```
